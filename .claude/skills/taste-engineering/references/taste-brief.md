# Project Taste Brief — isaisafacist.club

## Project Identity

### This project believes:
Friendship humor is best when it's overengineered to absurdity. A joke between friends deserves the production value of a government intelligence briefing. The gap between the gravity of the presentation and the stupidity of the content IS the art. Bureaucratic paranoia applied to mundane personal traits is peak comedy.

### This project is for:
Isa's friend group — people who get the inside jokes, appreciate absurdist humor, and will lose their minds at how far someone went with a dumb bit. The secondary audience is anyone who stumbles on it and has to figure out if it's real.

### This project should feel like:
- A declassified FBI file that someone clearly made as a joke but put TERRIFYING amounts of effort into
- Tim Robinson's "I Think You Should Leave" — the humor comes from commitment to the bit, not winking at the audience
- An SCP Foundation entry — the format IS the comedy, the more seriously it presents itself, the funnier it gets

### This project should NEVER feel like:
- A generic "roast your friend" website template
- A birthday surprise page with balloons and confetti
- Anything that breaks character or winks at the audience
- A tech demo showing off CSS skills (the skills serve the bit, never the reverse)

### The one word that MUST describe this project: COMMITTED
### The one word that must NEVER describe this project: CUTE

### Anti-References (projects whose DNA we reject):
- Generic "about me" joke pages
- Facebook birthday wall posts
- Any page that uses comic sans or "funny" fonts
- Joke sites that look intentionally bad

### Unexpected References (outside this space):
- Actual FOIA declassified documents — the typography, redactions, coffee stains, bureaucratic language
- SCP Foundation wiki entries — clinical tone describing absurd phenomena
- Tim Robinson sketches — the humor is in the COMMITMENT, not the punchline

### Taste Constraints:
- NEVER break character. The site is a real government document. Period.
- Every visual detail must reinforce the bureaucratic fiction (coffee stains, redactions, stamps, evidence frames)
- Inside jokes should be delivered with the gravity of intelligence findings
- Performance effects (glitch, scan lines, grain) serve the "leaked document viewed on an old terminal" fiction
- The site should make someone say "what the fuck" before they start laughing
